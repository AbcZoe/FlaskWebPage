需要安裝以下套件:
pip install flask_sqlalchemy
pip install flask_wtf
pip install email_validator
pip install flask_bcrypt
pip install flask_login
pip install Pillow
pip install flask-mail

需要清空資料庫裡的資料時:
from app import app, db
with app.app_context():
    db.drop_all()      # 刪掉舊表
    db.create_all()    # 重新建立表
